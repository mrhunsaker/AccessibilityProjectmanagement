# User Guide

Day-to-day usage of APM once it's installed: the dashboard, search, file
lineage, and the production workflows themselves.

| Page | Description |
|------|-------------|
| [Dashboard](dashboard.md) | Active jobs, low-stock alerts, recent activity, and quick create |
| [Quick Create](quick-create.md) | Creating any job type with minimal fields from the dashboard |
| [Search](search.md) | Global full-text search across jobs, files, metadata, and events |
| [Lineage Viewer](lineage-viewer.md) | Mermaid-based graph of file-to-job relationships |
| [Workflows](workflows/index.md) | Step-by-step guides for each production type |

For a specific production type (Braille, Large Print, eBraille, EPUB3/DAISY,
Tactile Graphics, 3-D Print), go to [Workflows](workflows/index.md).
