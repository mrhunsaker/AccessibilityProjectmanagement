# Large Print Workflow

**Guide to producing large print materials for visually impaired students.**

---

## 📋 Workflow Steps

### 1. **Intake & Requirements**
- **Purpose**: Gather specifications
- **Tasks**:
  - Confirm student's visual acuity
  - Determine required font size
  - Note color/contrast preferences
  - Check for specific formatting needs
- **Key Questions**:
  - What is the student's preferred font size?
  - Are there color/contrast requirements?
  - Any specific layout preferences?

### 2. **Source Preparation**
- **Purpose**: Optimize source document
- **Tasks**:
  - Convert to editable format if needed
  - Remove unnecessary formatting
  - Simplify complex layouts
  - Extract images for separate processing
- **Tools**: Pandoc, LibreOffice

### 3. **Formatting**
- **Purpose**: Apply large print styles
- **Tasks**:
  - Set font size (18pt, 20pt, 24pt, etc.)
  - Adjust line spacing (1.5x or 2x)
  - Increase margins
  - Simplify tables and lists
- **Font Recommendations**:
  - **Sans-serif**: Arial, Verdana, Helvetica
  - **Serif**: Times New Roman (for some students)

### 4. **Image Processing**
- **Purpose**: Make images accessible
- **Tasks**:
  - Increase resolution (300+ DPI)
  - Add alt text
  - Ensure sufficient contrast
  - Enlarge or simplify complex images
- **Tools**: GIMP, ImageMagick

### 5. **Proofreading**
- **Purpose**: Verify readability
- **Tasks**:
  - Check font consistency
  - Verify image quality
  - Test with screen magnifier
  - Review color contrast
- **Tools**: Screen magnifier, color contrast analyzer

### 6. **Printing**
- **Purpose**: Produce physical output
- **Tasks**:
  - Select printer
  - Load paper
  - Configure print settings
  - Run test print
- **Equipment**: Large format printer
- **Consumables**: Large print paper, toner/ink

### 7. **Binding & Finishing**
- **Purpose**: Prepare for delivery
- **Tasks**:
  - Collate pages
  - Bind or staple
  - Add cover page if needed
  - Package securely
- **Options**:
  - Spiral binding
  - Comb binding
  - Stapled
  - Loose leaf

### 8. **Delivery**
- **Purpose**: Send to recipient
- **Tasks**:
  - Record delivery method
  - Capture signature if required
  - Update inventory
- **Status**: Delivered

## 🎨 Formatting Guidelines

### Font Sizes by Grade Level
   Grade Level | Recommended Size | Minimum Size |
 |-------------|------------------|--------------|
 | K-2 | 24pt | 20pt |
 | 3-5 | 20pt | 18pt |
 | 6-8 | 18pt | 16pt |
 | 9-12 | 16pt | 14pt |
 | College | 14pt | 12pt |

### Spacing Requirements
 | Element | Standard | Large Print |
 |---------|----------|-------------|
 | Line spacing | 1.0x | 1.5x - 2.0x |
 | Paragraph spacing | 0pt | 6pt - 12pt |
 | Margins | 1" | 1.25" - 1.5" |
 | Column spacing | 0.5" | 0.75" - 1" |

### Color & Contrast
- **Minimum contrast ratio**: 4.5:1
- **Recommended**: Black text on white background
- **Acceptable alternatives**:
  - Dark blue on white
  - Black on light yellow
  - White on black (for some students)

## 🔧 Common Issues & Solutions
 | Issue | Cause | Solution |
 |-------|-------|----------|
 | **Text too small** | Wrong font size | Increase to minimum 18pt |
 | **Poor contrast** | Low contrast colors | Use high contrast theme |
 | **Images unclear** | Low resolution | Re-scan at 300+ DPI |
 | **Layout broken** | Complex formatting | Simplify layout |
 | **Paper jams** | Wrong paper size | Check printer settings |
 | **Toner smudging** | Low quality toner | Replace toner cartridge |
