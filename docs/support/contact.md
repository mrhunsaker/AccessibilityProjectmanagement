# Contact & Support

---

## 🐛 Bug Reports

Open a GitHub Issue:
<https://github.com/mrhunsaker/AccessibilityProjectManagement/issues>

Please include:
- APM version / git commit hash
- Python version
- Operating system
- Steps to reproduce
- Expected vs actual behaviour
- Relevant log output (run with `ACCESSMAN_LOG_LEVEL=DEBUG`)
